# 🎯 Advanced Sorting Algorithm Visualizer

A beautiful and feature-rich sorting algorithm visualizer built with Python, Tkinter, and Matplotlib.

## ✨ Features

- **6 Sorting Algorithms**: Bubble, Selection, Insertion, Merge, Quick, and Heap Sort
- **Beautiful UI**: Modern dark theme with color-coded visualizations
- **Real-time Statistics**: Track comparisons, swaps, and execution time
- **Time Complexity Analysis**: Visual graphs showing best, average, and worst cases
- **Customizable**: Adjust array size, value range, and sorting speed
- **Interactive**: Generate random arrays and watch algorithms in action

## 🚀 Installation

1. Install Python 3.7+
2. Install dependencies:
```bash
pip install matplotlib numpy
```

## ▶️ How to Run

```bash
python3 main.py
```

## 🎮 How to Use

1. **Select Algorithm**: Choose from dropdown menu
2. **Adjust Settings**: Set array size, min/max values, and speed
3. **Generate Array**: Click "Generate Array" button
4. **Start Sorting**: Click "Start Sorting" to visualize
5. **Analyze Complexity**: Click "Analyze Complexity" for graphs

## 📊 Algorithms Included

| Algorithm | Best | Average | Worst | Space |
|-----------|------|---------|-------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) |

## 🎨 Color Coding

- **Blue**: Default unsorted elements
- **Yellow**: Elements being compared
- **Green**: Sorted elements
